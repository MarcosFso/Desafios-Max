# Sistema de Loja em Python

## Conceitos utilizados

- Listas (armazenar produtos e vendas)
- Dicionários (representar dados)
- Funções
- Condicionais e loops
- Tratamento de erros (try/except)

## Funcionalidades

- Cadastro de produtos
- Realização de vendas
- Cálculo automático com desconto
- Relatório de vendas
- Salvamento em arquivo

## Regras

- Desconto de 5% para mais de 10 unidades
- Validação de estoque
- Produtos únicos
