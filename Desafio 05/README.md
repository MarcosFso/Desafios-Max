# Desafio 05 — Matrizes em Python

Implementação prática de criação, manipulação e operações com matrizes em Python, utilizando listas nativas e NumPy.

## Tecnologias
- Python 3.8+
- NumPy
- Pytest

## Instalação das dependências

```bash
pip install numpy pytest
```

## Execução

```bash
cd src
python main.py
```

## Testes

```bash
pytest tests/tests_matrizes.py -v
```
