# Operações com Listas em Python

Este projeto mostra exemplos básicos de como usar listas em Python.

## O que tem no código

* Criar listas
* Adicionar elementos (`append`, `insert`)
* Remover elementos (`remove`, `pop`)
* Modificar valores
* Concatenar listas
* Ver tamanho (`len`)
* Verificar elemento (`in`)
* Ordenar (`sort`)
* Inverter (`reverse`)
* Percorrer listas (`for` e `while`)
* Listas aninhadas
* List comprehension
* Uso de lista como pilha

## Como executar

1. Abra o arquivo `main.py` no VSCode
2. Execute o código

## Objetivo

Praticar operações básicas com listas em Python.
